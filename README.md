
# Loan Risk Analytics & Default Prediction System

---

## 🔍 Project Overview
This project analyzes customer credit data to identify risk patterns and predict loan default behavior.  
It combines SQL, Power BI, and Machine Learning to deliver end-to-end insights.

---

## 🎯 Objectives
- Analyze customer credit behavior and loan trends  
- Identify key factors influencing loan default  
- Segment customers into risk categories  
- Build a predictive model for default probability  
- Provide actionable insights for risk mitigation  

---

## 🧰 Tech Stack
- **SQL** → Data cleaning & transformation  
- **Power BI** → Dashboarding & visualization  
- **Python** → Data analysis & ML modeling  
- **Libraries** → Pandas, NumPy, Scikit-learn

## 🤖 Machine Learning Model

**Model Used:** Logistic Regression  

**Performance:**
- Accuracy: ~94%  
- AUC Score: ~0.91  

---

## 📈 Model Insights
- Strong performance for non-default prediction  
- Moderate recall for default cases  
- Delinquency features are most important  

---

## 💡 Key Insights
- Higher delinquency → higher default risk  
- Lower credit score → higher probability of default  
- Payment history is strongest predictor  

---

## 📌 Recommendations
- Focus on high delinquency customers  
- Apply stricter checks for low credit score  
- Improve model recall using class balancing
